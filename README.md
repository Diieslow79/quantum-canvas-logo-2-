
# Quantum Canvas Logo 2

**Live Demo:** Live Demo: https://diieslow79.github.io/quantum-canvas-logo-2


An interactive, generative logo for a digital art marketplace where every piece is co-created between human imagination and AI.

## Features

- **Generative** - New neural network pattern on every page load
- **Interactive** - Nodes respond to mouse movement with physics-based repulsion
- **Animated** - Orbital motion with pulsing connections and traveling particles
- **Responsive** - Adapts to any screen size
- **Zero Dependencies** - Pure HTML5 Canvas & vanilla JavaScript

## How to Use

1. **Move your cursor** - Nodes react and distort the network
2. **Click anywhere** - Instantly regenerate with new random connections
3. **Watch** - Particles travel along connections, nodes orbit and pulse

## Brand Concept

This logo embodies the "Generative Commerce" philosophy - just as the logo is never the same twice, every product in the Quantum Canvas store is uniquely generated at the moment of purchase.

## Tech Stack

- HTML5 Canvas API
- CSS3 Animations
- Vanilla JavaScript (ES6+)
- No external libraries

## Deployment

Hosted free on GitHub Pages. Simply fork this repo and enable Pages in settings to create your own variation.
